Belajar HTML, CSS, dan JavaScript

Materi Pembelajaran:

  • HTML: Pengenalan HTML 
    <!DOCTYPE html>
    <html>
    <head>
        <title>Halaman Pertama</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
    </html>
  • HTML: Struktur Dasar HTML 
    <header>Header</header>
    <nav>Menu Navigasi</nav>
    <main>Konten Utama</main>
    <footer>Footer</footer>
  • HTML: Tag dan Atribut HTML 
    <img src="gambar.jpg" alt="Deskripsi Gambar">
  • CSS: Pengenalan CSS 
    body {
        background-color: lightblue;
        color: white;
    }
  • CSS: Selector dan Properti CSS 
    h1 {
        font-size: 24px;
        color: blue;
    }
  • CSS: Flexbox dan Grid 
    .container {
        display: flex;
        justify-content: space-between;
    }
  • JavaScript: Pengenalan JavaScript 
    console.log("Hello, World!");
  • JavaScript: Variabel dan Tipe Data 
    let nama = "John";  
    let umur = 25;
  • JavaScript: Fungsi dan Event Handling 
    function sapa() {
        alert("Halo, selamat datang!");
    }
    document.getElementById("tombol").onclick = sapa;
  • JavaScript: DOM Manipulation 
    document.getElementById("judul").innerHTML = "Belajar JavaScript";
  • JavaScript: AJAX dan Fetch API 
    fetch('https://api.example.com/data')
        .then(response => response.json())
        .then(data => console.log(data));
  • JavaScript & Framework: Framework Frontend (React, Vue, dll.) 
    // React
    function App() {
        return <h1>Hello, React!</h1>;
    }

Belajar Cyber Security

Pelajari cara melindungi sistem dari ancaman digital.

  • Pengenalan Cyber Security: Konsep dasar dan pentingnya keamanan digital.
  • Jenis Ancaman: Malware, Phishing, DDoS, Man-in-the-Middle, dll.
  • Keamanan Jaringan: Firewall, VPN, IDS/IPS.
  • Keamanan Website: SQL Injection, XSS, CSRF, Clickjacking.
  • Keamanan Perangkat: Antivirus, Enkripsi, 2FA.
  • Keamanan Data: Hashing, Salting, Backup.
  • Ethical Hacking: Reconnaissance, Exploitation.
  • Praktik Keamanan: Password Management, Social Engineering.
  • Forensik Digital: Incident Response, Log Analysis.
  • Cyber Law: Hukum dan etika hacking.
